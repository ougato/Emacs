# Git命令安装

---

## Windows

> * git clone [https://github.com/ougato/Emacs.git]()
> * cp -a ./Emacs/.emacs.d/* C:/Users/Administrator/AppData/Roaming/.emacs.d

---

## Mac

> * git clone [https://github.com/ougato/Emacs.git]()
> * cp -a ./Emacs/.emacs.d/* ~/.emacs.d/

---

## Linux

> * git clone [https://github.com/ougato/Emacs.git]()
> * cp -a ./Emacs/.emacs.d/* ~/.emacs.d/
